前言：


day 0

由上可知，经过了 555 天的游学，我的情绪成功在联赛前炸到了谷底。

考虑到是 NOIp，给自己预估的及格线是 200pts。

直观感觉我要寄了，强忍着痛大概是在午夜后睡着了。

day 1

早上 7:30 才出门，被 cyx 怒喷太慢了，然后他就不等我先进考场了。8：00 的时候到了，碰到了 darko 但是没有碰到其他认识的人，我妈给我拍了张然后就也进去了。

发现 rdf 的键盘就是依托答辩，但是也只能将就着用。心情依然不是很好，就写了一堆小作文。考场非常傻逼，把 pdf 密码当成压缩包密码发给我们，导致 8：25 才解开压缩包，8：29 才看到题面。

开题发现 A 是傻逼，8：40 时过了 A，然后开 B 发现也是傻逼，但是中途发现我更傻逼，浪费了点时间大概是 9:30 才过的 B。

开 C 发现不会，简单思索了一下发现 O(qnm)O(qnm)O(qnm) 是容易的（但当时题看错了导致我以为是 252525pts），但是懒得写所以先开了 D。

看了眼题发现 O(kmin⁡{n,m})O(k\min\{n,m\})O(kmin{n,m}) 是容易想的，于是能过 A 性质及以前的所有点（这是个伏笔，因为其实中间有 121212 分是不行的……），然后 B 性质直接单调队列，C 性质就直接弃了（但是不难发现 B 性质和 C 性质是一样的……）。算了一下发现有 76 分（别问我咋算的，其实是 60 分……），加起来发现有 301301301pts，然后重新看了眼转移方程不知道为什么认为正解会是四边形不等式优化（悔恨。其实不难想到线段树优化的。但是我没想。没脑子选手就是这样的。），所以就懒得仔细想了先开写了。

然后出了点小锅就一直没调出来，11:00 的时候先把 C 的暴力补了，并开心地发现其实是 35pts 于是发现自己似乎是 311311311pts 然后就非常有自信了属于是。调了许久发现是离散化等一堆地方出锅了改了一下，在 12:30 时过了 A 性质及以前的点，认为自己是 303303303pts，但重新算了一遍发现其实是 299299299pts（可我没发现其实还算错了……）于是光速补了一下 B 性质，但是中途调的时候手残把关于相邻的判断删了，并且由于 ccf 的垃圾大样例所以没有发现，最后检查了一下各个文件就结束了。

于是刚出来的时候觉得自己是 100+100+35+72=307，还很有水平，面了下 caca、farfar 和 superyijin，但是 caca 发烧了而且要赶飞机，所以中午就我和 farfar 和 superyijin 一起吃的饭。聊着聊着才意识到原来 D 是容易的，悔不当初，但是也没什么办法。

但是在回家路上，看了眼群才意识到 D 的 B、C 性质做法是通用的！！！于是认为自己就 323323323pts 了，好像也能接受！！！然后回家开开电脑才发现。原来我把相邻判断删了导致这俩其实都挂了。只有 299299299pts 了，于是开始筹划紫砂，非常感谢当时的好几位同学把我劝下来了。

晚上熨斗数据出了，测了一下更开心了！！！这才意识到我还多算了 121212 分！！！我其实只有 287287287pts！！！人直接麻掉了！！！于是什么心情也没了，emo 了一晚上。但是考虑到我这一周都没怎么睡过觉，所以成功在 22:00 时由于困的受不了了就睡着了！！！

早上看小图灵给的是 283283283pts，不明白为什么但是反正全国排名直接快 600600600 了，很难绷。中午看就莫名其妙 299299299pts 了，但是排名也挺烂的，很乐。又得知 mmx C 过了总分 320+320+320+，而且据他所说学弟 colinxu 326326326，发现自己被单调队列了，多少有点难过。
